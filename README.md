# AI Interviewer App

A Flutter-based mobile app powered by ChatGPT to practice job interviews with text + voice input, multi-role support, payments, and admin dashboard.

## Features
- Login/Registration
- Text + Voice Interview Practice
- AI Answer Evaluation
- Premium Subscription (Razorpay)
- Admin Panel for Roles & Questions
- Notifications & Scheduling
- Multi-language support (English implemented)
- Unit, Widget, and Integration Tests

## Folder Structure
- lib/: Flutter source code
- test/: Unit, Widget, Integration tests
- android/, ios/: Platform-specific builds
- .github/workflows/: CI/CD pipeline
